## Overview

This is a complete authentication system with basic common functionality built using NEXT.js Framework with features like:
* User Login / Register / Logout
* Session management
* Secure Email verification
* Password Reset
* Sign in with Google
* Google reCAPTCHA Security

## How To Try it?
1. First, clone this repo ( make sure you have Node v22 installed )
2. Run the project using:
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

# NOTES
1. You have to add a suitable ```.env``` file in order for the project to work.
2. All main configurations are stored in the <b>settings.ts</b> file
3. If you face any problems, just create a pull request.
